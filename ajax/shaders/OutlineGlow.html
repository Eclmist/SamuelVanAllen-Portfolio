Outline Glow

Made as part of the game Snowfield.

This effect allows any objects to be highlighted with any color. As this is a image effect solution, it is easy to apply without any changes to existing object's materials.

Steps to render involves:

  • Render object to a separate buffer with a flat color replacement shader
  • Downsample and blur the render, then subtracting with the pre-blurred texture
  • Composit onto the normal scene render
