Pulse

Made as part of the game Wither.

Pulse also writes to the saturation mask described above using fragment to world space arithmetics.

This effect is achieved with 3 steps:

  • Calculate viewport position of starting point of pulse
  • Sample depth texture and calculate world position of each fragment using camera's position
  • Compare distance and write value to saturation mask accordingly. UV positions per fragment is also offset to produce a distortion effect along the edges of the pulse

A sphere collider also expands at the same rate and triggers code to make invisible objects glow.
