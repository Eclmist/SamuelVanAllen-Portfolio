Dissolve

Made as part of the game Wither.

Dissolve is a frament shader that masks the mesh based on perlin noise, and applies a color ramp to edges.

This effect is achieved with main 2 steps:

  • Fragments are discarded based on perlin noise
  • An edge ramp is sampled using the fragment's current value on the noise texture.

Unity's surface shader wrapper is used to enable forward shadows that account for discarded pixels.
