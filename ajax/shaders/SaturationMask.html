Stencil Saturation Mask

Made as part of the game Wither.

Stencil Saturation Mask is a screen space image effect that selectively desaturates parts of the rendered frame. In the game Wither, this effect is used to allow the player to spread colors from his feet.

This effect is achieved with 2 main steps:

  • Draw invisible stencil masks to the screen. (Particle Systems were used in the case of Wither, to allow for instancing)
  • Desaturate the rendered frame while sampling the stencil buffer

Some workarounds had to be used in Unity specifically, as the stencil buffer is cleared after the lighting pass. We can, in this case, render the mask to a seperate buffer. I described this issue (and a solution) here.
