DX12 Hybrid Rendererr

Ether is a 3D renderer written with DirectX 12 and C++, meant to be used as a platform for creating and testing modern graphics rendering techniques.

Some of the features Ether supports are:

  • Physically Based Rendering - HDR pipeline with physical light units
  • UE4 Microfacet BRDF, Image Based Lighting
  • Raytracing with DXR (GI, reflections, shadows)
  • Post Processing (Bloom, TAA, etc.)
  • GPU Resource/Descriptor Management
  • Shader Hot Reloading
  • Asset Pipeline /w Tool Integration Matcha Editor
